Instruções do programa

----------------------------------------------

Compilação:
para compilar digite : make sistema ou make all
para executar digite : ./sistema
para limpar os arquivos : make clean
compilar em outro c++ diferente do 11 : make all dialect=C++14 ou make all dialect=C++17

----------------------------------------------

Opções do menu:

0.Fechar programa

1.Criar e adicionar Atleta.

2.Procurar Atleta pelo nome.

3.Imprimir arvore completa.

----------------------------------------------

Tipos de Atleta :

Atleta.
AtletaVar (este possui idade em suas informações)

----------------------------------------------
