// Resumo Programa:
O programa possui as disciplinas padrões :
Matematica
Portugues
Filosofia
Almoso
Comp

E a partir disso requere a inserção do numero de periodos e alunos.
Com isso possibilita o acesso de estatisticas dos alunos e disciplinas com uso das notas a partir das opções oferecidas.
(as notas de cada aluno são criadas aleatoriamente).

Funções :
1)Retorna a média movel do periodo n de cada disciplina

2)Representa de forma agrupada as disciplinas que tiveram melhora/neutralidade/piora no desempenho de notas

3)Representa o desempenho individual de cada aluno

4)Retorna a melhor disciplina com melhores notas finais

5)Mostra o melhor aluno do periodo atual

6)Mostra a disciplina que teve a maior evolução no ultimo periodo
