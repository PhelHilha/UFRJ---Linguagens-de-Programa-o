Instruções do programa

Compilação:
para compilar digite : make
para executar digite : ./programa (numero da opção)
para limpar os arquivos : make clean

Opções do menu:

1.Imprime na tela a lista de caminhos(arestas) de cada Estado

2.Imprime na tela a Matriz[i][j] com a distancia(peso) de cada caminho(aresta)

3.Exibe na tela o custo minimo de um Estado para o outro(caminho com menor peso)

4.Imprime na tela a densidade do Grafo

5.Imprime na tela o Estado(vertice) com maior centralidade
