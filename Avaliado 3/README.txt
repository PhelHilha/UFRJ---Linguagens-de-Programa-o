Instruções do programa

----------------------------------------------

Compilação:
para compilar digite : make
para executar digite : ./programa
para limpar os arquivos : make clean

----------------------------------------------

Opções do menu:

0.Fechar programa

1.Criar e adicionar disciplina ao historico.

2.Adicionar historico anterior ao atual.

3.Imprimir historico completo.

4.Buscar e imprimir disciplina especifica.

5.Imprimir CRA do Aluno.

6.Deletar disciplina do historico.

7.Editar nota de disciplina.

----------------------------------------------

Dados.txt formatação:

NomeDaDisciplina Periodo Peso NotaFinal

----------------------------------------------
